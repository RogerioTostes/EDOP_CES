Descrição do Projeto:

Esse projeto é uma oportunidade para aplicar estruturas de dados e algoritmos, a equipe deu enfase em escolher um algoritmo que não
havia sido ensinado nos parametros da sala de aula e outro que, ja havia sido visto na disciplina de Estruturas de Dados I.
Consiste na implementacao de uma arvore de decisao, utilizando o algoritmo Minimax para um jogo do tipo deterministico, o Jogo da Velha
ou o Tic Tac Toe.
As estruturas da dados utilizadas foram:
1) Pilha - Presente no desenvolvimento do algoritmo em JS
2) Arvore de Busca - Presente em ambos os desenvolvimentos
3) Principio do Algoritmo Minimax para Busca Competitiva.

As linguagens utilizadas foram HTML, CSS, JS, JQuery, algumas API do Google Chrome e da Font Awesome, bem como C#.

As IDE utilizadas foram:
1) Sublime Text ++ 3
2) Visual Studio
